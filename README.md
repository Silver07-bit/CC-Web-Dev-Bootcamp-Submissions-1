## 🚀 Welcome to the User Form App Project!

Hey everyone 👋
This is our Web Development Bootcamp Day 2 project repo. Follow the steps below to get started.

---

## 📥 Clone the Repository

Open your terminal and run:

```bash
git clone https://github.com/Ashitha0409/DAY2_CC_bootcamp.git
```

## 🌐 Run the App

Open `index.html` in your browser or use a live server extension in VS Code. (can also press Run from the top bar)

The app includes:
- A user form to submit name and email
- Image gallery
- Sum calculator for practicing JS functions

---

## 💡 Notes

* Make sure you have **Git installed**
